# Contributing

## 浊莲内部成员

> 在执行以下操作之前，请确保你已被第五议会邀请并加入浊莲 Github 的组织仓库。

1. 克隆此仓库
2. 新建分支：

    ```bash
    git branch new_branch_name
    git checkout new_branch_name
    ```

3. 完成修订
4. 保存并推送：

    ```bash
    git add *
    git commit -m ":sparkles: 新增功能"
    git push
    ```

5. 发起 Pull Request

## 外部合作者

1. Fork 此仓库
2. 新建分支：

    ```bash
    git branch new_branch_name
    git checkout new_branch_name
    ```

3. 完成修订
4. 保存并推送：

    ```bash
    git add *
    git commit -m ":sparkles: 新增功能"
    git push
    ```

5. （可选）完成合并：

    ```bash
    git checkout main
    git pull origin main
    git merge new_branch_name
    ```

6. 发起 Pull Request
